# Experiments

Each experiment should state its hypothesis, methodology, data, validation method, metrics,
results, and conclusion. Start from the Obsidian experiment template before running work.

Phase 7 generated result artifacts belong under `results/`, not this directory. Each result
manifest hashes its configuration, normalized event stream, and feature dataset; it must also
state source fidelity, truth categories, latency, fill-model assumptions, and limitations.
